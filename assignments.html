Fanpage Project

Using HTML and CSS create a fanpage for anything you want. It could be as traditional as a celebrity page, something as mundane as a chair, or something as specific as a page devoted to cats riding roombas.

Requirements

  • At least 4 separate pages
  • Use styles in a .css file
  • Must include media of some sort on each page (images, video, audio)
  • Host on github

p5 Project

Using a combination of variables, images, for loops, animations, conditionals, statements, text, color, etc. create an application that uses keyboard and mouse user input. This can be an animation, a simple game, a drawing machine type app.

Requirements

  • One for loop
  • Two conditional statements
  • Mouse input
  • Keyboard input
  • Upload to a github respository
  • Add at least one still image and a description to your github repository readme

Web Narrative Project

Using p5, HTML, and CSS create an interactive narrative web app that uses interactive elements such as sliders, buttons, color, clicks, text boxes, etc. The narrative can be whatever you want, from something as ordinary as a commute, to something imagined like an alien invasion, or a narrative that endlessly loops to mess with a viewers idea of time and space.

For the project:

  • Create a minimum of 5 different 'scenes' that are triggered by different interactions
  • Use a minimum of 3 different HTML input elements to progress the narrative, e.g. buttons, sliders, links, selects, etc.
  • Every scene should have an interactive element on the page
  • Upload to a github respository
  • Add at least one still image and a description to your github repository readme

Advanced p5 Project

Using p5, p5 libraries, and JavaScript libraries make a more complex interactive application. Utilize one of the libaries discussed in class or another you are interested in to create an application that moves beyond the browser - user relationships we have made so far (webcam, audio, machine learning, etc).

Proposal

A typed 1-2 page proposal with a drawing, sketch, mockup due via email. In your proposal discuss what you would like to work on for your final project, why you want to produce this, and your conceptual reasoning behind wanting to create the work. Include a breakdown of the steps to complete your project and an approximate timeline for completing the steps. Each student will meet with me individually to discuss their proposal and to get feedback.

Final Project

Drawing inspiration form artists we have viewed in class and using any of the techniques we have covered in the course, create a more developed project. This can be a significant expansion of a previous project or a completely new project. Your project can be a game, an interactive narrative, an interactive 'installation', a webcam app, a data driven app, or a combination of any of these. This is an opportunity to connect your personal interests with the course content in a way that is interesting to you.

You can use whatever software you want, as long as it adheres to the following guidelines:

  • Dynamic functionality: use callbacks, booleans, conditionals, and interactivity (mouse, keyboard, webcam, etc.) to create a dynamic project. The key here is to use HTML/CSS, JavaScript (p5) to trigger levels, scenes, aesthetics, etc.
  • Use at lease one piece of media (image, video, sound)
  • Submit a link of your github repository before critique date
